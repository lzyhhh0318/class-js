# 课程资源与作业管理子系统 - 项目整合说明

## 📦 项目结构

```
ai-time-scheduler/
├── src/                      # 前端源代码
│   ├── components/          # 通用组件
│   ├── pages/              # 页面组件
│   │   ├── Home.tsx        # 首页
│   │   ├── Materials.tsx   # 课程资料页
│   │   ├── Assignments.tsx # 作业管理页
│   │   ├── AIParse.tsx     # AI解析页
│   │   └── ...
│   ├── store/              # Zustand 状态管理
│   ├── assets/             # 静态资源
│   ├── App.tsx             # 应用入口
│   └── main.tsx            # React 挂载点
├── api/                    # 后端 API 代码 (Express)
│   ├── routes/            # API 路由
│   │   ├── materials.ts   # 资料管理 API
│   │   ├── assignments.ts # 作业管理 API
│   │   ├── ai.ts          # AI 相关 API
│   │   └── ...
│   ├── app.ts             # Express 应用
│   └── server.ts          # 服务器入口
├── public/                # 静态资源
├── dist/                  # 前端打包产物 (已构建)
├── package.json           # 依赖配置
├── tsconfig.json          # TypeScript 配置
├── vite.config.ts         # Vite 构建配置
├── tailwind.config.js     # TailwindCSS 配置
├── postcss.config.js      # PostCSS 配置
└── .env                   # 环境变量示例
```

## 🚀 快速开始

### 1. 安装依赖
```bash
npm install
```

### 2. 配置环境变量
复制 `.env` 并根据实际情况修改：
- OSS 服务器地址
- AI API 密钥
- 端口配置等

### 3. 启动开发服务
```bash
# 同时启动前端和后端
npm run dev

# 或者分别启动
npm run client:dev  # 前端 (http://localhost:5173)
npm run server:dev  # 后端 (http://localhost:3001)
```

### 4. 构建生产版本
```bash
npm run build
```

## 🔧 核心功能模块

### 1. 课程资料管理 ([Materials.tsx](src/pages/Materials.tsx))
- 课件上传与分类
- 资料预览与下载
- 版本管理
- 资料删除

### 2. 作业管理 ([Assignments.tsx](src/pages/Assignments.tsx))
- 作业发布
- 学生作业提交
- 教师批改与评分

### 3. AI 解析 ([AIParse.tsx](src/pages/AIParse.tsx))
- 课件智能解析
- 知识点提取
- AI 问答

### 4. 首页 ([Home.tsx](src/pages/Home.tsx))
- AI 助教卡片
- 快捷操作入口

## 🔌 API 接口说明

### 基础路径
- 前端开发: `http://localhost:5173`
- 后端 API: `http://localhost:3001/api`

### 主要接口
| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/materials` | 获取资料列表 |
| POST | `/api/materials/upload` | 上传资料 |
| DELETE | `/api/materials/:id` | 删除资料 |
| GET | `/api/assignments` | 获取作业列表 |
| POST | `/api/assignments/submit` | 提交作业 |
| POST | `/api/ai/qa` | AI 问答 |

## 🔐 注意事项

1. **环境变量**: 部署前务必配置正确的 `.env` 文件
2. **OSS 代理**: 根据实际网络环境调整 `vite.config.ts` 中的代理配置
3. **静态文件**: `uploads/` 目录需要有写权限
4. **数据库**: 当前使用内存存储，生产环境建议接入真实数据库
